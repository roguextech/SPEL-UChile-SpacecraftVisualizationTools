# SpacecraftVisualizationTools

This tool was designed to visualize * .CSV data obtained from the spacecraft simulator [Here](https://github.com/spel-uchile/SpacecraftSimulator).


### Required Libraries
 - Numpy
 - jpl-ephem
 - PyQt5
 - threading
 - pyvistaqt
 - pyquaternion
 - pyqtgraph
 - pyvista
 
### Installation

pip install -r requirements.txt
